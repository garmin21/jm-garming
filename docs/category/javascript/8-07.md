### 1. debounce 防抖函数

顾名思义，防止在一段时间内，持续抖动，（程序中，解释，防止一个函数，或一段代码持续性的执行）


### 2. 手写一个简单版的防抖函数

```js
// 用法
document.onmousemove=debounce(changeNum,500)

function debounce(fn, delay) {
    let timer = null;
    return (...args) => {
        if(timer) clearTimeout(timer)
        timer = setTimeout(() => {
            fn(...args)
        }, delay);
    }
}
```
