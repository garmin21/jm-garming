-   用于生成 svg 雪碧图
    https://github.com/anncwb/vite-plugin-svg-icons

-   使用 gzip 或者 brotli 来压缩资源。
    https://github.com/anncwb/vite-plugin-compression

-   提供本地和生产模拟服务
